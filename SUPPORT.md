# Suporte

Para ajuda geral, usa GitHub Issues.

Antes de abrir uma issue:

- Confirma que instalaste dependencias no `backend` e `frontend`.
- Corre `npm run build` a partir da raiz.
- Ve se ja existe uma issue parecida.

Inclui sempre:

- Sistema operativo.
- Versao de Node.js.
- Comando executado.
- Erro completo, sem chaves ou tokens.
